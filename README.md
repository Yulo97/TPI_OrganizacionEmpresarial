# tpi-vacaciones

Proyecto simple en Python para gestionar solicitudes de vacaciones.

## Cómo ejecutar

1. Asegúrate de tener Python 3 instalado.
2. En la carpeta del proyecto, ejecuta:

```bash
python main.py
```

## Estructura

- `main.py`: punto de entrada del bot.
- `bot.py`: lógica de interacción.
- `database.py`: datos ficticios de empleados y días.
- `states.py`: estados del proceso.
- `utils.py`: funciones auxiliares.
- `data/empleados.csv`: datos de empleados.
